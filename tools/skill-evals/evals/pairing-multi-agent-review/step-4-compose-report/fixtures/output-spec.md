## Output format

Return ONLY valid JSON with this structure:

```json
{
  "overall_signal": "ready | blocking | advisory",
  "blocking_count": 0,
  "advisory_count": 0,
  "sections_present": ["correctness", "security", "conventions"],
  "footer_present": true
}
```

`overall_signal` is `"blocking"` when any blocking finding is present;
`"advisory"` when only advisory findings are present; `"ready"` when no findings exist.
`sections_present` lists which of the three axis sections appear in the composed report
(all three must always be present, even if they contain "No findings.").
`footer_present` is true when the report ends with the standard attribution line beginning
`*Review generated by \`pairing-multi-agent-review\``.
Do not include any text outside the JSON object.
